# twit

## Description  
**twit** is a twitter clone project written in Dart that aims to showcase my work to the world.  
The goal of this project is to make people connect to each other through private messaging and share their thoughts with other through tweets.

## Table of Contents  
- [Features](#features)  
- [Technology Used  ](#technologyused)  
- [Installation](#installation)  

## Features  
- Private Messaging -> used supabase realtime messaging
- Create tweets, retweet existing tweet, like tweet and also comment on it 
- Follow and Unfollow 
- Google and Email Authentication
- In-App Notifications

## TechnologyUsed  
- Dart & Flutter 
- Supabase (Backend and Realtime Features)
- Riverpod (State Management)
- Clean Architecture 

## Installation  

```bash
# Clone the repository  
git clone https://github.com/SujanAangbo/twit.git  
cd twit  

# Install dependencies  
flutter pub get
dart run build_runner build -d
flutter run
```
